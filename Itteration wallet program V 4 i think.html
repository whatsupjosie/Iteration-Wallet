Introduction
Welcome to Iteration Wallet.

Before you protect anything real, spend two minutes here.

There are three commands — OPEN, REMOVE, DELETE.
You will practise each one on a file that does not exist.

Nothing destructive can happen without you typing the word yourself,
one character at a time, in your own time.

Nothing moves inside the Vault until you unlock it. Type OPEN to unlock the practice vault below.

practice_document.txt ● Protected
OPEN
Type it below — no paste, no autofill
✓ Vault opened. The file is now accessible.

REMOVE takes the file out of the Vault into a holding area. It still exists. You can still change your mind. Type REMOVE.

practice_document.txt ● Protected
REMOVE
Type it below — character by character
✓ File moved to holding. Still recoverable.

DELETE is permanent. A file must be REMOVEd before it can be deleted — two deliberate decisions, not one. This practice file is not real. Nothing is actually deleted.

practice_document.txt ● Holding
DELETE
Type it — feel the weight of it
✓ Done. You know how the Vault works.

Three commands. That is the complete system.

OPEN — unlock the Vault for a session
REMOVE — move a file to holding
DELETE — permanently remove (requires REMOVE first)

Nothing happens to your files until you decide it should.
Not an AI tool. Not a sync service. Not ransomware. Nothing.
